Per partner:
Upload DON-hosted secret JSON for that partner (or versioned update), e.g.:
  { "AUTHORIZATION": "Bearer <partner_token>" }

Chainlink gives/uses slotId + version.

Configure router:
consumer = partner proxy address
enabled = true
slotId/version = partner’s values
Router then attaches those secrets to that partner’s request only.

brownie run scripts/configure_partner_auth.py main 0xRouter 0xPartnerProxy true 1 1 --network sepolia
Args:
1) router_address = Chainlink consumer contract
2) partner_proxy_address = partner caller identity for auth secrets mapping
3) enabled
4) slotId
5) version
If omitted, script can read from .env:
CORE_RATE_ROUTER_ADDRESS
PARTNER_PROXY_ADDRESS
